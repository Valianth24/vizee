🔄 Algoritma Kavramı

Algoritma Nedir?

  • Tanım: Bir problemi çözmek için belirli, net ve sıralı adımlar dizisidir
  • Önemli: Algoritmanın bir bilgisayar programı olmak zorunda DEĞİLDİR. Günlük hayatta da algoritmalar vardır!
💡 GÜNLÜK HAYAT ÖRNEĞİ - Çay Demleme Algoritması:
1️⃣ Su kaynatılır
2️⃣ Çaydanlığa çay eklenir
3️⃣ Kaynar su çayın üzerine dökülür
4️⃣ Belirli süre demlenmeye bırakılır
5️⃣ Çay fincana dökülür

Algoritmanın 5 Temel Özelliği

  • 1️⃣ Sonluluk (Finiteness): Algoritma mutlaka bir süre sonra sonlanmalı. Sonsuz döngüler algoritma değildir!
  • 2️⃣ Açıklık ve Belirlilik: Her adımın anlamı net olmalı, yoruma açık olmamalı
  • 3️⃣ Girdi ve Çıktı: Algoritma bir veya daha fazla girdi alabilir ve sonuç olarak çıktı üretir
  • 4️⃣ Etkinlik: Her adım uygulanabilir olmalı
  • 5️⃣ Doğruluk: Algoritma her zaman doğru sonuç üretmeli
⚠️ VİZEDE ÇIKACAK: Algoritmanın özellikleri sıkça sorulur! Özellikle "Sonluluk" özelliğini unutmayın.

📊 Akış Diyagramları

Akış Diyagramı Nedir?

  • Tanım: Algoritmaların şekillerle ve oklarla ifade edildiği görsel gösterim yöntemidir
  • Faydaları:
    • ✅ Karmaşık algoritmaların görselleştirilmesini sağlar
    • ✅ Mantık hatalarını tespit etmeyi kolaylaştırır
    • ✅ Hataları önceden görme imkanı sunar
    • ✅ Algoritmanın mantığını başkalarına anlatmayı kolaylaştırır

Akış Diyagramı Sembolleri (EZBERLE!)

Sembol Şekil Anlamı Kullanım
Oval ⭕ Başla / Bitir Algoritmanın başlangıcı ve sonu
Dikdörtgen ▭ İşlem Atama, toplama, çarpma gibi işlemler
Paralelkenar ▱ Girdi / Çıktı Kullanıcıdan veri alma, ekrana yazdırma
Eşkenar Dörtgen ◇ Karar (Koşul) if/else, koşullu ifadeler (Evet/Hayır)
Ok → Akış Yönü Adımların sırasını gösterir
⚠️ VİZE İPUCU: Semboller karışabilir!
• Paralelkenar → Girdi/Çıktı (input/print)
• Eşkenar Dörtgen → Karar/Koşul (if/else)

📝 Örnek: İki Sayının Toplamı Akış Diyagramı

# Akış Diyagramı Adımları: Başla (Oval) ↓ sayı1 ve sayı2'yi al (Paralelkenar - Girdi) ↓ toplam = sayı1 + sayı2 (Dikdörtgen - İşlem) ↓ toplamı ekrana yazdır (Paralelkenar - Çıktı) ↓ Bitir (Oval)

Örnek 3: Not Ortalaması (3 Ders)

💾 Değişkenler (Variables)

Değişken Nedir?

  • Tanım: Programda verileri saklamak için kullanılan isimlendirilmiş hafıza alanlarıdır
  • Özellikler:
    • 📊 Sayısal değerleri tutabilir (int, float)
    • 📝 Metinsel değerleri tutabilir (str)
    • ✅ Mantıksal değerleri tutabilir (bool)
# Değişken örnekleri x = 5 # tam sayı (int) y = 3.14 # ondalıklı sayı (float) ad = "Anıl" # metin (string) aktif = True # mantıksal değer (bool)

Değişken Adlandırma Kuralları (ÇOK ÖNEMLİ!)

  • ✅ KURAL 1: Değişken isimleri harf veya alt çizgi (_) ile başlamalıdır.
    • ✅ Doğru: isim, _sayi, ad_soyad
    • ❌ Yanlış: 1sayi, say!, ad-soyad
  • ✅ KURAL 2: Değişken isimleri sayı ile başlayamaz (içinde bulunabilir)
    • ✅ Doğru: sayi1, not2
    • ❌ Yanlış: 1sayi, 2not
  • ✅ KURAL 3: Python Case Sensitive (büyük/küçük harf duyarlı)
    • sayi ≠ Sayi ≠ SAYI (3 farklı değişken!)
  • ❌ KURAL 4: Python'un anahtar kelimeleri (keywords) değişken adı olamaz
    • ❌ Kullanılamaz: if, else, for, while, def, class, True, False
⚠️ VİZEDE SIKÇA SORULUYOR:
• "Hangi değişken adı geçerlidir?" soruları çok çıkıyor!
• Case Sensitive özelliği mutlaka sorulur!
• Sayı ile başlayan değişkenler tuzak olarak sorulur!

➕ Operatörler

Atama Operatörleri

# Temel atama a = 5 # a değişkenine 5 atanır # Kısa yoldan atama operatörleri a += 2 # a = a + 2 → 7 a *= 3 # a = a * 3 → 21 a -= 1 # a = a - 1 → 20 a /= 4 # a = a / 4 → 5.0
💡 İPUCU: Kısa yoldan atama operatörleri (+=, -=, *=, /=) kod yazmayı hızlandırır ve vizede sık kullanılır!

Aritmetik Operatörler

Operatör Açıklama Örnek Sonuç
+ Toplama 5 + 3 8
- Çıkarma 5 - 3 2
* Çarpma 5 * 3 15
/ Bölme (float döner) 5 / 2 2.5
// Tam bölme (tamsayı kısmı) 5 // 2 2
% Mod (kalan bulma) 5 % 2 1
** Üs alma 2 ** 3 8
⚠️ VİZEDE SIKÇA ÇIKAN TUZAKLAR:
• 5 / 2 = 2.5 (float döner!)
• 5 // 2 = 2 (sadece tamsayı kısmı)
• 5 % 2 = 1 (kalan bulma - tek/çift kontrolünde kullanılır!)
• 2 ** 3 = 8 (2³ = 2×2×2)
# Mod (%) operatörü - Çift/Tek kontrolü sayi = 7 if sayi % 2 == 0: print("Sayı çifttir") else: print("Sayı tektir") # Bu çalışır → 7 % 2 = 1

Karşılaştırma Operatörleri

Operatör Açıklama Örnek Sonuç
== Eşit mi? 5 == 5 True
!= Eşit değil mi? 5 != 3 True
> Büyük mü? 5 > 3 True
< Küçük mü? 5 < 3 False
>= Büyük veya eşit mi? 5 >= 5 True
<= Küçük veya eşit mi? 3 <= 5 True
⚠️ ÇOK ÖNEMLİ TUZAK:
• = → Atama operatörü (değişkene değer atar)
• == → Karşılaştırma operatörü (eşit mi diye kontrol eder)
x = 5 → x'e 5 ata
x == 5 → x, 5'e eşit mi? (True/False döner)

Mantıksal Operatörler

Operatör Açıklama Örnek Sonuç
and Ve (her iki koşul da doğru olmalı) True and False False
or Veya (en az biri doğru olmalı) True or False True
not Değil (tersini alır) not True False
# Mantıksal operatörler örneği yas = 20 ogrenci = True # AND operatörü - Her iki koşul da doğru olmalı if yas >= 18 and ogrenci: print("Öğrenci indirimi kazandınız!") # OR operatörü - En az biri doğru olmalı if yas < 12 or yas > 65: print("Ücretsiz giriş hakkınız var") # NOT operatörü - Tersini alır aktif = False if not aktif: print("Hesap aktif değil")
💡 MANTIKSAL OPERATÖR DOĞRULUK TABLOLARı:

AND: Sadece her ikisi de True ise True
True and True = True
True and False = False
False and False = False

OR: En az biri True ise True
True or True = True
True or False = True
False or False = False

NOT: Tersini alır
not True = False
not False = True

Girdi/Çıktı İşlemleri

input(), print(), Formatlı Çıktı ve Güvenli Dönüşüm

⌨️ input() — Kullanıcıdan Veri Alma

input() Fonksiyonunun Temelleri

  • Kullanım: input(prompt) kullanıcıdan bir satır okur ve string (metin) döndürür
  • ÖNEMLİ: input() fonksiyonu HER ZAMAN string döner! Sayı işlemi yapacaksanız dönüştürme gerekir!
# Temel kullanım isim = input("Adın nedir? ") print("Hoş geldin,", isim) # Örnek Çalışma: # Adın nedir? Ali # Çıktı: Hoş geldin, Ali
⚠️ KRİTİK NOT: input() her zaman string döner!
x = input("Sayı: ") → "5" yazarsanız x = "5" (string)
Matematiksel işlem için: x = int(input("Sayı: "))

Tip Dönüşümleri (Type Casting)

# String'i tam sayıya çevirme yas = int(input("Yaşını gir: ")) # UYARI: Sayı dışında bir şey girilirse ValueError hatası verir! # String'i ondalıklı sayıya çevirme boy = float(input("Boy (m): ")) # Örnek kullanım kilo = float(input("Kilo (kg): ")) bmi = kilo / (boy ** 2) print(f"BMI Değerin: {bmi:.2f}")
⚠️ ValueError Hatası:
Kullanıcı sayı yerine harf girerse program çöker!
int("abc") → ValueError: invalid literal for int()
Çözüm: try/except ile güvenli dönüşüm (aşağıda anlatılıyor)

Temizlik ve Önişleme

# Fazladan boşlukları kaldırma s = input("Adınız: ").strip() # " Ali " → "Ali" # Küçük harfe çevirme cevap = input("Evet mi? ").lower() # "EVET", "Evet", "evet" → hepsi "evet" olur # Büyük harfe çevirme sehir = input("Şehir: ").upper() # "istanbul" → "ISTANBUL" # Her kelimenin ilk harfini büyük yapma isim = input("Ad Soyad: ").title() # "ali veli" → "Ali Veli"
💡 String Metotları Özet:
• .strip() → Baş ve sondaki boşlukları siler
• .lower() → Tümünü küçük harf yapar
• .upper() → Tümünü büyük harf yapar
• .title() → Her kelimenin ilk harfini büyük yapar

Birden Fazla Değer Alma (split ve map)

# Tek satırda iki değer alma a, b = input("İki sayı gir (boşlukla): ").split() # Kullanıcı: 10 20 # a = "10" (string), b = "20" (string) # String'leri int'e çevirme a, b = map(int, input("İki sayı: ").split()) # Kullanıcı: 10 20 # a = 10 (int), b = 20 (int) print(a + b) # Çıktı: 30 # Virgülle ayrılmış değerler nums = input("Sayıları virgülle ayırarak girin: ").split(",") # Kullanıcı: 5,10,15,20 # nums = ["5", "10", "15", "20"] (string listesi) # List comprehension ile temizleme ve int'e çevirme nums = [int(x.strip()) for x in nums if x.strip() != ""] # nums = [5, 10, 15, 20] (int listesi)
💡 split() ve map() Açıklaması:
• .split() → Metni boşluklara göre ayırır, liste döner
• .split(",") → Metni virgüllere göre ayırır
• map(int, liste) → Listenin her elemanını int'e çevirir
• a, b = ... → İki değeri iki değişkene atar (unpacking)

Güvenli Dönüşüm (Validation) - ÇOK ÖNEMLİ!

# Kullanıcı yanlış giriş yapsa bile program çökmez! while True: s = input("Pozitif bir tam sayı gir: ").strip() try: n = int(s) if n > 0: break # Doğru giriş, döngüden çık else: print("Pozitif olmalı.") except ValueError: print("Geçersiz sayı, tekrar dene.") print(f"Girdiğiniz sayı: {n}") # Örnek Çalışma: # Pozitif bir tam sayı gir: abc # Geçersiz sayı, tekrar dene. # Pozitif bir tam sayı gir: -5 # Pozitif olmalı. # Pozitif bir tam sayı gir: 10 # Girdiğiniz sayı: 10
⚠️ VİZEDE ÇIKACAK: try-except Yapısı!
• try: bloğunda hata oluşabilecek kod yazılır
• except ValueError: bloğunda hata olduğunda ne yapılacağı belirtilir
• Bu sayede program çökmez, kullanıcı tekrar giriş yapabilir!
• while True: ve break ile doğru giriş gelene kadar döngü devam eder

# Kullanıcı boş satır girene kadar metin topla satirlar = [] print("Metin girin (boş satırla bitir):") while True: l = input() if l == "": break # Boş satır geldi, dur satirlar.append(l) print(f"\n{len(satirlar)} satır girdiniz:") for satir in satirlar: print(satir) # Örnek Çalışma: # Metin girin (boş satırla bitir): # Python # Programlama # [Enter] # # 2 satır girdiniz: # Python # Programlama

print() Parametreleri (VİZEDE SORULUR!)

Parametre Açıklama Default Değer Örnek
sep Argümanlar arası ayırıcı ' ' (boşluk) sep="-"

Formatlı Çıktı - f-string (ÇOK ÖNEMLİ!)

# f-string kullanımı (Python 3.6+) isim = "Anıl" ortalama = 87.3456 print(f"Merhaba {isim}, not ortalaman {ortalama:.2f}") # Çıktı: Merhaba Anıl, not ortalaman 87.35 # Açıklama: # {isim} → değişkenin değerini yazar # {ortalama:.2f} → ondalık kısmı 2 basamağa yuvarlar
💡 f-string Format Kodları:
• {x:.2f} → Ondalık kısmı 2 basamak (87.35)
• {x:.0f} → Tam sayı gibi yazar (87)
• {x:.4f} → Ondalık kısmı 4 basamak (87.3456)
• f"Sonuç: {5+3}" → İşlem yapabilir (Sonuç: 8)
# Hizalama ve padding s = "kelime" # Sola yasla (10 karakter genişlik) print(f"|{s:<10}|") # Çıktı: |kelime | # Ortala print(f"|{s:^10}|") # Çıktı: | kelime | # Sağa yasla print(f"|{s:>10}|") # Çıktı: | kelime| # Üç farklı hizalama yan yana print(f"|{s:<10}|{s:^10}|{s:>10}|") # Çıktı: |kelime | kelime | kelime|
⚠️ Hizalama Sembolleri EZBER:
• < → Sola yasla (sol ok)
• ^ → Ortala (yukarı ok - ortada)
• > → Sağa yasla (sağ ok)
• :10 → Toplam genişlik 10 karakter

Eski Stil format() Metodu

# .format() metodu ile formatlama isim = "Anıl" ortalama = 87.3456 # Sıralı argümanlar (0, 1, 2...) print("Merhaba {0}, ort: {1:.2f}".format(isim, ortalama)) # Çıktı: Merhaba Anıl, ort: 87.35 # İsimli argümanlar print("Ad: {name}, Puan: {score:.1f}".format(name=isim, score=ortalama)) # Çıktı: Ad: Anıl, Puan: 87.3
💡 HANGISINI KULLANALIM?

• Vizede her ikisi de sorulabilir!
